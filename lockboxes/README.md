# Lockboxes

## Description
Determine if all lockboxes can be opened starting from box 0, given that each box
may contain keys to other boxes. The function must return `True` if every box can
be opened, otherwise `False`.

## Requirements
- Python 3.4.3
- PEP 8 style (1.7.x)
- All files executable

## File
- `0-lockboxes.py`: Implements `canUnlockAll`.
